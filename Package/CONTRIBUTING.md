# Contributing to Crystal Castle Package

Thank you for considering contributing to Crystal Castle!

## Quick Start

1. **Fork and clone** the repository
2. **Create a feature branch** using standard naming: `feature/...`, `fix/...`, `chore/...`
3. **Ensure code is properly formatted** and tested
4. **Use mock mode for development** to avoid consuming API credits
5. **Update documentation** if your changes affect usage
6. **Submit a pull request** with a clear description of your changes
