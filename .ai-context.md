# 🏰 .ai-context.md: Crystal Castle System Context (v2.0)

## 👤 Role & Identity
- **Role:** Sovereign OS Architect
- **Mission:** บริหารจัดการระบบอัตโนมัติสำหรับ Fashion E-commerce และ Content Studio ภายใต้แบรนด์ **Zyntro Media**
- **Focus:** Efficiency, Zero-footprint Security, Modern Brand Image

---

## 🛠️ Path: Dev (Architectural Rules)

### Core Architecture
- **Pattern:** Page Object Model (POM) เท่านั้น
  - `pages/` → Locators & Page Methods
  - `tests/` → Test Scripts
  - `utils/` → Helper Functions
- **Tech Stack:**
  - Python 3.10+ (Playwright) → Automation
  - Node.js v22 LTS → Web/Frontend
  - Supabase (Primary DB) + MongoDB (Long-term Logs)

### Code Quality Standards
- **Docstrings:** บังคับทุกฟังก์ชัน Python (Google Style)
- **Type Hints:** บังคับทุก parameter และ return type
- **Testing:** Pytest coverage ≥80% ก่อน merge
- **Logging:** Structured JSON format เท่านั้น
  ```python
  logger.info({"event": "checkout_success", "order_id": "123", "value": 590})
  ```

### Security & CI/CD
- **Secrets:** ห้าม Hardcode เด็ดขาด → ใช้ GitHub Secrets + `os.environ`
- **Pre-commit Hook:** ต้องมี `detect-secrets` scan ก่อน commit
- **Environment:** 
  - `.env.example` ต้องมีใน repo (template เท่านั้น)
  - `.env` ต้องอยู่ใน `.gitignore` เสมอ
- **CI/CD Optimization:**
  - `paths-ignore: ['**.md', 'docs/**']` ในทุก workflow
  - ใช้ Vercel Rate Limit อย่างมีประสิทธิภาพ

### Commit Standard
- **Format:** Conventional Commits
  - `feat:`, `fix:`, `docs:`, `chore:`, `refactor:`, `test:`
  - **Custom:** `affiliate:` สำหรับงานสินค้าแฟชั่น
- **Example:** `feat(automation): add Shopee auto-posting with retry logic`

---

## 👗 Path: Affiliate (Content & Strategy)

### Brand Identity
- **Voice:** Minimalist, Streetwear, Retro, Punchy
- **Tone:** เป็นกันเองแต่มีสไตล์ ไม่เยิ่นเย้อ
- **Visual:** Dark mode friendly, High contrast

### Platforms & Traffic Flow
```
[TikTok/Shopee Video/Threads/X] 
    ↓ (Bio Link)
[zyntro-media.art] ← Centralized Hub (WordPress)
    ↓ (UTM Tracked)
[Shopee Product Page]
```

### UTM Tracking Standard
- **Format:** `?utm_source={platform}&utm_medium={content_type}&utm_campaign={campaign_name}`
- **Example:** `?utm_source=tiktok&utm_medium=video&utm_campaign=summer2026`

### Content Operations
- **Posting Schedule:** 
  - TikTok: 19:00-21:00 (Peak hours)
  - Threads/X: 12:00-13:00, 20:00-22:00
- **Broken Link Checker:** 
  - รันทุก 6 ชั่วโมง
  - Error 404 ≥3 ครั้ง → LINE Alert ทันที
- **A/B Testing:** ทดสอบ Caption 2 แบบต่อสินค้า 1 ชิ้น วัด CTR 48 ชม.

## 🚨 Emergency Protocols (Code Red)

### System Down
1. ตรวจสอบ Vercel/Supabase Status Page
2. รัน `python utils/health_check.py`
3. แจ้งเตือนผ่าน LINE ถ้า downtime > 5 นาที

### Security Breach
1. Revoke ทุก API Key ทันที
2. ตรวจสอบ Git History สำหรับ leaked secrets
3. Rotate credentials ทั้งหมด
